ESPECIFICACIONES RECOMENDADAS:

Logos:
  - Formato: PNG con transparencia
  - Tamaño: 300x80px (banorte-logo.png)
  - Tamaño: 300x80px (banorte-logo-white.png)

Recompensas/Ahorros/Challenges:
  - Formato: PNG o JPG
  - Tamaño: 400x400px (cuadrado)
  - Optimizadas para web (< 200KB)

Green Credit:
  - Formato: JPG
  - Tamaño: 800x600px (landscape)
  - Alta calidad (< 500KB)

Education:
  - Formato: PNG o JPG
  - Tamaño: 600x400px (landscape)
  - Optimizadas para web (< 300KB)

Backgrounds:
  - Formato: JPG
  - Tamaño: 1920x1080px (Full HD)
  - Alta calidad (< 1MB)
  - Usar herramientas de compresión (TinyPNG, ImageOptim)

- Freepik.com (gratis con atribución)

NOTAS:
